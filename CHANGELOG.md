# YFonts 更新日志

本文件记录 YFonts 面向用户的重要功能变化。

## 1.21.0 - 2026-06-24

### macOS 桌面体验

- 增加 macOS 原生标题栏、交通灯位置和深浅外观同步，统一顶部区域的系统视觉。
- 增加中文原生应用菜单，提供导入、同步、搜索、字体视图、分类管理、侧栏、主题、设置、更新和 GitHub 入口。
- 右键菜单、项目包菜单、导入菜单和字重面板采用 macOS 半透明材质，并修复浅色模式的悬停对比度。
- 支持 Apple Silicon 与 Intel Mac 的 Universal 应用和 DMG 打包。

### 界面与操作

- 将筛选改为语言、授权、来源三组单击快捷按钮，移除多层下拉操作并强化分组层级。
- 将库管理入口移动到侧边栏底部，保留 macOS `Command+,` 和 Windows `Ctrl+,` 快捷键。
- 将版本与作者信息调整为低视觉权重的 `LYZ · v1.21.0`。
- 优化 macOS 字体、焦点、圆角、标题栏留白和侧边栏折叠体验。

### 交互修复

- 修复“查看字重和文件”需要先选中字体或点击两次才能打开的问题。
- 修复拖拽字体到项目包时意外框选页面文字的问题。
- 增加符合平台习惯的项目包删除与重命名键盘操作。
- 首次启动时跟随系统深浅外观，并同步桌面窗口主题。

### 跨平台构建

- Windows 与 macOS 继续共用同一项目和核心界面，保留各自原生标题栏与交互差异。
- 增加 Windows NSIS、macOS 本地 DMG、macOS Universal DMG 构建命令。
- 增加 GitHub Actions 双平台发布流程；正式 macOS 公共发布可接入 Developer ID 签名与 Apple 公证。

## 1.20.1 - 2026-06-15

### 应用更新

- 修复新版本发布后仍显示旧 GitHub Release 信息的问题。
- 每次打开库管理面板时主动刷新最新版本，不再受旧的六小时缓存影响。
- 更新请求禁用网络缓存，并增加时间戳避免中间缓存返回旧数据。
- 主动检查失败时明确显示错误；非主动检查仍可使用缓存作为断网兜底。

## 1.20.0 - 2026-06-15

### 本地字体管理

- 完善文件夹与单个字体文件导入，支持多个独立字体来源并保留真实路径。
- 改进字体元数据、文件夹结构和语言字符集识别，减少中文字体被误分到英文分类、类型目录被识别为字体名称的问题。
- 合并同一家族的字重、斜体和不同文件格式，减少相同前缀字体的重复预览。
- 支持手动修正字体语言、风格分类和授权状态，并可调整分类显示顺序。
- 完善收藏、隐藏、移出、恢复、重复字体整理及批量管理流程。
- 改进项目字体包的创建、分类、拖放和键盘操作。

### 字体预览

- 使用真实字体文件渲染中文和英文预览。
- 支持静态字重切换、统一字重预览轴和可变字体轴。
- 默认优先展示接近中等字重的样式，并在切换字重时同步更新列表预览。
- 增加最近常用字体定位，改进大量字体下的虚拟列表性能与预览稳定性。

### 在线发现

- 接入 Google Fonts 与 Fontsource 在线字体目录。
- 在线字体、在线数量和来源状态与本地字体库完全分离，不计入全部字体、本地字体包或免费商用统计。
- 支持在线来源切换、授权信息、预览与下载入口。
- 增加断网、请求失败和缓存数据提示，本地字体管理不依赖网络。

### 桌面应用与界面

- 使用 Tauri 构建 Windows 桌面应用，支持打开字体真实位置和原生文件选择。
- 统一明暗主题、控件、右键菜单和桌面标题栏视觉。
- 更新 YFonts 程序图标与开发者信息。
- 优化侧边栏、筛选标签、详情面板和窄窗口布局。
- 增加 GitHub 版本检查入口。

### 数据与隐私

- 安装包不包含开发者字体、字体路径、项目包、收藏或个人库数据。
- 字体索引和用户设置保存在当前用户的应用数据目录中。
